## 2.控制板->上位机回应询问

| 2-1  | 字节数（24） | 命令信息           | 释义                              |
| ---- | ------------ | ------------------ | --------------------------------- |
| CMD  | 4            | 0XAA55AA10         | 回应询问                          |
|      | 6            | 板卡 MAC 地址      | 得到板卡MAC地址                   |
|      | 4            | 板卡 IP 地址       | 板卡 IP 地址                      |
|      | 1            | 符号位信息         | 无符号数 0x00: 有符号数   0x01    |
|      | 1            | 每通道有效数据长度 | 默认12位，即0x0B。如果是8位则0x07 |
|      | 4            | 当前采样率         | 单位Hz                            |
|      | 4            | 每次发送数据字节数 | 16KB=16bit*4*512*4                |
| 2-2  | 字节数（4）  | 命令信息           | 释义                              |
| CMD  | 4            | 0XAA55FFF1         | 开发板正常采集状态回应            |
